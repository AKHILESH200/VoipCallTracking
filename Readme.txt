------Tool under development------(for now Only on Windows)

Installing Requirments:
------------------------- 
1. Download and Install Wireshark (https://www.wireshark.org/#download)
2. Copy Path of "tshark" from installed folder of Wireshark and update in Environment Variables
3. Install the required python libraries with the help of command in CMD (opened in the path of VoIP_Tracer_v1.0):
	python -m pip install -r requiments.txt
(Optional) 4. To make the tool to run from any location update the path in Environment Variables 

Usage:
--------
Open CMD or Powershell, RUN using command: python vtrac.py
